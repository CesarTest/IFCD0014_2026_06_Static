Web Site Design!

Implementing a system based on Wordpress shortcodes easy enough to allow people not familiar with the internet to have his own site where they can publish easily enough through Windows Live Writer very rich contents (movies, music, documents and images). By reducing the usage of database to the least using shortcodes, web services and fix multimedia folder for images, I try to avoid future mantainance problems.

Browsers: IE7 won't render fine, due to IE image processing bug. Please use IE8, Opera 9+, Firefox 3+ or Chrome.
